# Trampoline

Idle-кликер для Яндекс Игр. Игрок прыгает на батуте, зарабатывает монеты и постепенно прокачивает персонажа.

## Геймплей

- **Батут** — прыгаешь, зарабатываешь монеты за высоту полёта. Чем выше — тем больше
- **Тренажёрный зал** — качаешь силу, она автоматически растёт и увеличивает высоту прыжка
- **Магазин батутов** — 10 уровней, от «Старого батута» до «Лунного»
- **Крылья** — 12 уровней, ускоряют полёт
- **Питомцы** — гача-система с яйцами четырёх редкостей (обычное / редкое / эпическое / легендарное), питомцы дают множитель силы (до 3 активных одновременно)

## Стек

- **Unity 6000.3.14f1** (URP)
- **Платформа:** Яндекс Игры (WebGL)
- **Плагин:** [PluginYourGames v2](https://github.com/forcepusher/com.bananaparty.yandexgames) — авторизация, лидерборды, монетизация, локализация, хранилище
- **Input System** 1.19.0
- **TextMesh Pro**
- **Newtonsoft JSON** 3.2.2

## Запуск

1. Открыть в Unity **6000.3.14f1**
2. Открыть сцену `Assets/_Game/Scenes/SampleScene.unity`
3. Play — в редакторе работает в режиме симуляции (без реального SDK Яндекса)

## Сборка для Яндекс Игр

1. **File → Build Settings** → платформа **WebGL**
2. В настройках PluginYourGames выбрать платформу **YandexGames**
3. Build — плагин автоматически встраивает шаблон и SDK
